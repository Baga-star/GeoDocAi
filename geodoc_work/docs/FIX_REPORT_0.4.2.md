# GeoDoc AI 0.4.2 — delete, map preview and MapReader answer fix

## Исправлено

1. **Удаление документов**
   - Исправлена CORS-конфигурация: `DELETE` теперь разрешён в preflight.
   - Endpoint `DELETE /api/documents/{document_id}` проверен через FastAPI TestClient.

2. **Белый пустой экран в карточке карты**
   - Preview для PDF-rendered pages теперь обрезает near-white поля перед отправкой в UI.
   - Карта должна занимать карточку, а не выглядеть как маленький рисунок внутри пустого листа.

3. **Плохие заголовки карт**
   - Vision/OCR-фразы вида `карта: жёлтая заливка образует...` больше не используются как title.
   - Для таких случаев UI/backend возвращают нормальный заголовок: `Структурная карта по кровле пласта`.

4. **Повтор текста под картой**
   - Visual tab больше не повторяет один и тот же текст как заголовок и описание.
   - Вместо этого показывается блок `Что прочитал MapReader` с короткими bullet-пунктами.

5. **Ответ по карте**
   - Map answer переписан в единый профессиональный формат:
     - краткий вывод;
     - основная интерпретация;
     - зоны и легенда;
     - контуры/изолинии;
     - скважины/точки;
     - источник;
     - ограничения только при необходимости.
   - После вопроса UI теперь открывает вкладку `Ответ`, а не сразу raw-карточку карты.
